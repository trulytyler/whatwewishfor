1) Filters

  <li class="filter" data-filter="all">All</li>
  <li class="filter" data-filter="graphic">Graphic</li>
  <li class="filter" data-filter="photography">Photography</li>
  <li class="filter" data-filter="branding">Branding</li>
  <li class="filter" data-filter="video">Video</li>
</ul>
<!-- End Portfolio Filters -->

If you want to change the filters you just need to change the text and the 'data-filter' attribute.

2) Grid

<!-- Portfolio Grid -->
<ul id="portfolio-grid">
  <li class="mix photography branding mix_all">
    <img src="images/portfolio/1.jpg">
    <a href="project-1.html" class="open-project">
      <div class="project-overlay">
        <i class="fa fa-plus-circle fa-3x"></i>
        <span class="project-name">Silk Light</span>
        <span>Photography / Branding</span>         
      </div>
    </a>
  </li>
</ul>
<!-- End Portfolio Grid -->

Each element is composed of:

  • '<li>': To work with the filters every '<li>' tag must have the classes mix, mix_all and the filter class (one or more), for example photography and branding
    • Project image: Located in images/portfolio folder. Important: insert images with the same height, otherwise the grid may not work well.
    • '<a>': The 'href' attribute of this tag must link to an existing html file, in this case project-1.html
      • 'project-overlay': contains name and category of the project and an icon to expand it.

main.js

This is the main javascript file, you can make your changes here.

parallax-bg.js

Is called only in the Full Width Parallax Slider version. Don't include it if you don't want the slideshow.

Simple script to set a slider of images as background of the Home section.

$(document).ready(function(){
	//Home Image Slider
	$.backstretch([
	  "images/bg/1.jpg",
	  "images/bg/2.jpg",
	  "images/bg/3.jpg"    
	], {duration: 4500, fade: 'slow'});
	
});

You can define images, duration and effect. If you want you can add more images.

video-bg.js

Is called only in the Video Background version. Don't include it if you don't want the video background.

Use a Youtube video as background of the Home section.

$(document).ready(function(){
  //Video Background
  $(".player").mb_YTPlayer();
});

To work fine, this script needs an '<a>' tag with the 'player' class and the . Like this:

<a id="bgndVideo" class="player" data-property="{videoURL:'Lxdfg0drTcw',containment:'#video-wrapper',autoPlay:true, mute:true, startAt:0, opacity:1}">My video</a>

In the 'data-property' you can configure the video.

  • videoURL - You must enter the last part of the url of the youtube video, i.e for this http://www.youtube.com/watch?v=Lxdfg0drTcw, you need to enter Lxdfg0drTcw
  • containment - The container of the video, don't forget to call it '#video-wrapper'
  • autoplay - If is set to true, the video will start automatically.
  • mute - You can decide whether to enable the audio of the video
  • startAt - You can decide which minute should start the video.
  • opacity - The opacity on the video

Google Maps Configuration

In the main.js file, you can define your latitude and longitude. To set them, do the following:

Open the main.js file, in the /js/ folder, and go to the line 150:

var myLatlng = new google.maps.LatLng(40.773328,-73.960088); // <- Your latitude and longitude

Replace '40.773328,-73.960088' with your latitude and longitude (without spaces). You can find them here

Contact Form Configuration

The contact form is submitted without page refresh (AJAX), you just need to set your email address in the contact.php file.

$to = "email@yourdomain.com"; # <--- Your Email

Replace email@domain.com with your email address. That's it.

FAQ

Hot to change the color scheme?

If you want to change the color of the theme you just need to go in the 'head', uncomment the line that contains the color you want and comment the line with the color you want to remove.

For example, if you want the blue instead of the red, you would do the following:

    <!-- Color Scheme -->
    <!-- <link rel="stylesheet" type="text/css" href="css/colors/red.css"> -->
    <link rel="stylesheet" type="text/css" href="css/colors/blue.css"> 
    <!-- <link rel="stylesheet" type="text/css" href="css/colors/green.css"> -->
    <!-- <link rel="stylesheet" type="text/css" href="css/colors/black.css"> -->  
    <!-- <link rel="stylesheet" type="text/css" href="css/colors/yellow.css"> -->

How to change the version of the home section?

This theme has three different styles for the home section, ready for use:

  • Full Width Parallax Slider, named 'index.html'
  • Pattern Background, named 'pattern-bg.html'
  • Video Background, named 'video-bg.html'

In almost all hosting providers, the main page must be named index.html. So, if you want to use the Pattern Background or Video Background version, you need to rename it to index.html, but first, remember to rename the existing 'index.html' in parallax-bg.html, to not lose the Full Width Parallax Slider version.

How to change the icons?

All icons are provided with Font Awesome, their structure is similar to the following:

<i class="fa fa-user"></i>

If you want to change an icon on this theme, just find it at this link, and, in this case, replace the 'fa-user' class, with the class of the icon you want.

How to add or remove animations to elements?

If you want to add animation to an element, you need to add to it the classes "animated" and "hiding" and set the 'data-animation' attribute.

  • The animated class add the animation to the element.
  • The hiding class make the element invisible until the animation is performed.
  • The data-animation is the name of the animation you want to use.

For example:

<div class="promo animated hiding" data-animation="fadeInRight">
...
</div>

This example add the fadeInRight animation to the element.

If you want to animate a set of element one at a time, you need to add the 'data-delay' attribute. For example:

<div class="service animated hiding" data-animation="fadeInUp" data-delay="0">
...
</div>
<div class="service animated hiding" data-animation="fadeInUp" data-delay="500">
...
</div>
<div class="service animated hiding" data-animation="fadeInUp" data-delay="1000">
...
</div>
<div class="service animated hiding" data-animation="fadeInUp" data-delay="1500">
...
</div>

With this example, every 500 milliseconds an element is animated.

To remove animation from an element, you just need to remove the classes "animated" and "hinding" and remove the "data-animation" attribute.
